Sass: CSS with superpowers

Sass is the most mature, stable, and powerful professional grade CSS extension language in the world. Stylesheets are getting larger, more complex, and harder to maintain. This is where a CSS pre-processor can help. Sass lets you use features that do not exist in CSS, like variables, nested rules, mixins, imports, inheritance, built-in functions, and other stuff.
